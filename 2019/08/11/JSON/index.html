JSONObject是一个核心类，它内部利用HashMap进行存储，实现了从一个字符串、Map对象、Java对象，JSONObject对象自身来构建一个key-value形式的JSON数据格式类型对象（这种数据格式与JavaScript的对象格式非常相似）。从而达到Map一样的操作，并可以输出为标准的JSON字符串。
